## Build
`go build main.go`

## Run
Run the executable and make sure all these environment variables are available:
```
REAL_DEBRID_API_KEY=
OVERSEERR_HOST=
OVERSEERR_TOKEN=
PLEX_HOST=
PLEX_TOKEN=
PLEX_TV_ID=
PLEX_MOVIE_ID=
```
PLEX_X_ID= is for the id of the corresponding library in plex
